# E-Commerce Product Card UI

A responsive frontend built with **React, TypeScript, and Vite**, deployed on Vercel.

## Features
- **Responsive Design:** Product cards adapt seamlessly to mobile and desktop screens using Flexbox.
- **Dynamic Data:** Consumes the backend API to display real-time product info.
- **Category Filter:** Filter products by "Apparel" or "Electronics".
- **Cart Management:** Local state tracks the number of items added to the cart.
- **Stock Validation:** "Add to Cart" button is disabled if the product is out of stock.

## Tech Stack
- React 18 + TypeScript
- Vite
- CSS Modules / Inline Styles (Zero external UI libraries for portability)

## Local Setup
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
